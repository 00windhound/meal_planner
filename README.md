# meal_planner
This project will help people track daily nutrients, total food costs, and plan the most effective meals for their goals.
currently still in the planning phase.
user should be able to add ingredients, build recipies with those ingredients, and build a weekly or by weekly plan with the recipies. they can see what meals need covered, what ingredients are still unused, total cost so far. home page ingredients and meals should indicate needing to buy, have, and used up.
